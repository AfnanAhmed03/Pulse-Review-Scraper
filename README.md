# RetailPulse

RetailPulse is an AI-powered retail analytics platform that helps businesses understand customer behavior, predict churn, forecast demand, and optimize inventory management.

## Features

### Customer Segmentation

* KMeans Clustering
* Customer classification into business segments
* Interactive dashboard visualization

### Churn Prediction

* Customer churn analysis
* Churn rate monitoring
* XGBoost-based predictive modeling

### Demand Forecasting

* Sales forecasting using Prophet
* Seasonal trend analysis
* Future demand estimation

### Inventory Optimization

* Reorder point calculation
* Inventory status monitoring
* Stock replenishment recommendations

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* Prophet
* Streamlit
* Plotly

## Dataset

Online Retail Dataset containing 397,884 cleaned transactions and 4,338 customers.

## Project Architecture

Online Retail Dataset → Data Cleaning → Customer Segmentation → Churn Prediction → Demand Forecasting → Inventory Optimization → Streamlit Dashboard
