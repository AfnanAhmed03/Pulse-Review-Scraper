# SaaS Review Scraper

## Overview
This script scrapes product reviews from SaaS review platforms based on:
- Company name
- Date range
- Source (G2, Capterra, TrustRadius)

The output is a structured JSON file.

## Supported Sources
- G2
- Capterra
- TrustRadius (Bonus)
